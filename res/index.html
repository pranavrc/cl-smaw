this.onloop

Search and lookup artists, albums and tracks on Spotify.

Search

this.onloop.net/tracks/track name (Example: this.onloop.net/tracks/torn and frayed)
this.onloop.net/albums/album name (Example: this.onloop.net/albums/exile on main street)
this.onloop.net/artists/artist name (Example: this.onloop.net/artists/the rolling stones)

Lookup

this.onloop.net/track/track name or spotify URI (Example: this.onloop.net/track/torn and frayed)
this.onloop.net/album/album name or spotify URI (Example: this.onloop.net/album/exile on main street)
this.onloop.net/artist/artist name or spotify URI (Example: this.onloop.net/artist/the rolling stones)
The track, album and artist names can contain additional information, and a !(number) at the end to toggle relevancy. this.onloop.net/tracks/tumbling dice!1 gives you the most relevant result, !2 the second most, and so on. Default is !1.

Album Reviews

this.onloop.net/reviews/album name (Example: this.onloop.net/reviews/exile on main street)
Try it: this.onloop.net/
