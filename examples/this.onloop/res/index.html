this.onloop

Search and lookup artists, albums and tracks on Spotify.

Search

s.onloop.net/tracks/track name (Example: s.onloop.net/tracks/torn and frayed)
s.onloop.net/albums/album name (Example: s.onloop.net/albums/exile on main street)
s.onloop.net/artists/artist name (Example: s.onloop.net/artists/the rolling stones)

Lookup

s.onloop.net/track/track name (Example: s.onloop.net/track/torn and frayed)
s.onloop.net/album/album name (Example: s.onloop.net/album/exile on main street)
s.onloop.net/artist/artist name (Example: s.onloop.net/artist/the rolling stones)
The track, album and artist names can contain additional information, and a !(number) at the end to toggle relevancy. s.onloop.net/tracks/tumbling dice!1 gives you the most relevant result, !2 the second most, and so on. Default is !1.

Album Reviews

s.onloop.net/reviews/album name (Example: s.onloop.net/reviews/exile on main street)
Try it: s.onloop.net/
